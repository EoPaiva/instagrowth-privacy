# InstaGrowth IA uPaiva

Extensão Chrome para crescimento orgânico no Instagram com IA. Posicionamento: ferramenta de growth profissional, não bot barato.

> Cresça com estratégia, não no escuro.

## Recursos

- **Descobrir Perfis** — captura seguidores, curtidores, comentadores, hashtags e localizações
- **Fila de Crescimento** — Manual / Automático com carrossel ao vivo e timing humano
- **Growth Coach IA** — hashtags, tendências, plano semanal, rotina diária, análise de perfil, gerador de texto
- **Prompt Studio IA** — gerador de prompts para ChatGPT/Claude/Gemini/Midjourney/CapCut/Canva/etc
- **Premium** — license key com PIX direto via Mercado Pago

## Stack

- **Manifest V3**
- **esbuild** — bundle IIFE para popup/content/background
- **Backend Railway** compartilhado entre apps uPaiva (`appId: "instagrowth"`)
- **BYOK** (Bring Your Own Key) para IA — usuário cola sua chave de Claude/OpenAI/Gemini
